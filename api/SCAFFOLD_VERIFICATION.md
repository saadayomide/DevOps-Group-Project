## Quick Start

1. **Install dependencies**:
   ```bash
   cd api
   pip install -r requirements.txt
   ```

2. **Run the application**:
   ```bash
   python run.py
   # OR
   uvicorn app.main:app --reload
   ```

3. **Verify endpoints**:
   - Health: http://localhost:8000/health
   - API Docs: http://localhost:8000/docs
   - Root: http://localhost:8000/
